# Facial-Expression-with-Python

<img width="1268" height="970" alt="Screenshot 2025-07-28 at 19 07 16" src="https://github.com/user-attachments/assets/f7eac5ba-b3e1-499f-9cdf-e8d3014fb921" />


The model detects the facial expression through a camera and based on the detected expressions, some  Activities would be recommended. 

The expressions are: Happy, Angry, Sad, Exciting and Surprising. 

**Activities recommended:** 

1.Listening to songs
2. Watching movies 

They are recommended based on the expressions detected.

**Dataset:** 
Haarcasecade Classifier
Paul Ekman's Basic Emotions

**Packages used for the project:** 

OpenCV
Matplotlib
Numpy
Scipy
Scikit-learn
